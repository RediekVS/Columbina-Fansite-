# Columbina-Fansite-
Columbina Fansite es un sitio web funcional desarrollado con HTML y tecnologías web modernas. Destaca por su estética visual sólida y definida, con diseño llamativo y navegación fluida. Integra comunidad y sistema de subida de imágenes. Creado por Keider Ortega y potenciado con inteligencia artificial.

Sitio de fans de Columbina

Columbina Fansite es un sitio web funcional enfocado en una identidad visual fuerte y definida. Permite a los usuarios interactuar dentro de una comunidad y compartir imágenes a través de un sistema integrado de subida de contenido.

Características

Diseño visual llamativo y coherente.

Navegación fluida

Sistema de comunidad

Integración con servicios en la nube
